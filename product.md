Votre projet est une application administrative pour Yamdata, une plateforme qui permet aux utilisateurs d'acheter des forfaits internet tout en épargnant automatiquement.
Objectifs principaux :
Créer un tableau de bord administratif pour gérer les utilisateurs, les transactions et les statistiques
Développer une interface publique attrayante pour présenter le concept de Yamdata
Intégrer l'authentification via Supabase pour sécuriser l'accès administratif
Technologies utilisées :
Next.js comme framework principal
Tailwind CSS pour le design
Supabase pour la base de données et l'authentification
Zustand pour la gestion d'état
Structure du projet :
/app - Structure principale de l'application Next.js
/app/admin - Section administrative avec tableau de bord
/app/page.tsx - Page d'accueil publique
/components - Composants réutilisables (sidebar, header, etc.)
/lib - Utilitaires, configuration Supabase, stores Zustand
La page d'accueil présente actuellement Yamdata avec des sections pour les fonctionnalités, le fonctionnement et un exemple concret d'utilisation. Le tableau de bord administratif permet de gérer les utilisateurs et de visualiser les statistiques d'utilisation.
Les prochaines étapes pourraient inclure l'amélioration du design, l'ajout de fonctionnalités administratives supplémentaires et l'optimisation de l'expérience utilisateur.
